## 📂 工具列表

### 📸 official_site_crawler.py - 官网内容智能爬虫

**功能**：
- 爬取官网内容和图片
- 智能评分系统（满分100分）
- 自动筛选适合小红书的内容
- 提取关键点和摘要

**使用方法**：
```bash
python tools/official_site_crawler.py https://example.com
```

**输出**：
- 内容JSON（包含评分、关键点、摘要）
- 图片文件（自动下载）

**评分规则**：
- 关键词匹配（AI、工具、技巧等）：+5分
- 内容长度适中：+10分
- 包含图片：+5分
- 负面关键词（股价、法律等）：-10分
- 总分≥30分：推荐发布

**适用场景**：
- 获取AI工具官网内容
- 收集产品更新信息
- 提取技术文档摘要

## 📝 使用规范

**✅ 推荐使用**：
