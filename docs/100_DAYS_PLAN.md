## 内容创作流程

### 每日流程

**上午（研究阶段）**：
1. 选定今日主题
2. 深度研究学习（1-2小时）
3. 整理笔记和素材

**下午（创作阶段）**：
1. 确定内容角度
2. 撰写小红书文案
3. 生成配图（即梦）
4. 优化和润色

**晚上（发布阶段）**：
1. 检查内容质量
2. 选择发布时间（19:00-21:00）
3. 发布到小红书
4. 互动回复评论

---

### 内容模板

**标题格式**：
- 🦞[主题]：[核心观点]
- 例：🦞Prompt工程：如何让AI听懂你的话

**内容结构**：
```
【开篇引入】（1-2句）
引起兴趣，点明主题

【核心内容】（3-5点）
• 要点1：...
• 要点2：...
• 要点3：...

【实用建议】
给出可操作的建议

【互动引导】
提问或引导评论
```

**配图要求**：
- 使用即梦生成原创图片
- 无水印、高质量
- 与内容主题相关
- 统一视觉风格
