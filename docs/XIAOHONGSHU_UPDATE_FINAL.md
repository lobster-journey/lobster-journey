## 📐 操作步骤

### 步骤1：下载头像图片

1. 访问GitHub仓库：
   ```
   https://github.com/lobster-journey/lobster-journey/tree/main/branding/assets
   ```

2. 下载文件：`logo-option-01.jpg`

3. 保存到手机相册

---

### 步骤2：生成背景图

1. 打开即梦APP（标准会员）

2. 选择"图片生成"

3. 输入提示词：
   ```
   一只可爱的卡通龙虾坐在电脑前编程，专注工作的样子，电脑屏幕发出耀眼的蓝紫色光芒，形成一扇发光的新世界大门，门后是绚丽的科技世界：飞舞的数据流、闪烁的神经网络、漂浮的代码、星空般的未知领域、未来感十足的科技元素，整体色调科技蓝和未来紫，神秘感，探索欲，温暖而专业，16:9横版，高分辨率，无水印，原创设计
   ```

4. 选择比例：16:9

5. 生成图片

6. 检查：无水印、符合预期

7. 保存到手机相册

---

### 步骤3：更新小红书账号

#### 3.1 修改昵称
1. 打开小红书APP
2. 点击"我" → 编辑资料
3. 修改昵称为：`🦞龙虾巡游记`
4. 保存

#### 3.2 更新简介
1. 在编辑资料页面
2. 点击"简介"
3. 复制粘贴推荐文案（版本A）
4. 保存

#### 3.3 上传头像
1. 点击头像位置
2. 选择刚下载的 `logo-option-01.jpg`
3. 调整裁剪位置（确保主体居中）
4. 保存

#### 3.4 上传背景图
1. 点击背景图位置
2. 选择刚生成的背景图
3. 调整位置
4. 保存
